## ⚙️ 기술 스택

### 👩‍💻 백엔드
- Java 17  
- Spring Boot 3.5.3  
- Spring Web / Spring Data JPA  
- Spring Security  
- Lombok / Validation  
- MySQL  
- Swagger (API 문서화)  
- Postman (API 테스트)

- Google Gemini API (리뷰 요약 기능)
